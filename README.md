# Base limpa de API em Node

Este projeto serve exclusivamente para oferecer uma base rápida para criação de APIs em Node.

Ele possui apenas a configuração básica do Express com Typescript e o primeiro router.

## Instalação
Após descompactar o projeto rodar:

```bash
npm install
```

Depois copiar o arquivo `.env.example` para `.env.local`:

```bash
cp .env.example .env.local
```
E alterar as variáveis de ambiente.

## Uso
Para rodar o projeto, utilize o comando padrão:

```bash
npm run dev
```
